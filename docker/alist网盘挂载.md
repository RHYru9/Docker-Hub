###  alist网盘挂载
```
docker-compose.yaml
```
```
services:
  alist:
    image: xhofe/alist:latest
    container_name: alist
    restart: always
    volumes:
      - ./alist:/opt/alist/data
    ports:
      - "54321:5244"
    environment:
      - PUID=0
      - PGID=0
      - UMASK=022
```


##### 修改密码

```
docker exec -it alist ./alist admin set 你的密码
```

进入方式为：`ip:54321`

用户名为：`admin`

端口可自行修改

管理路径为`/@manage`
